School Management System

A comprehensive School Management System designed to simplify and manage day-to-day school operations, including student management, teacher management, staff management, attendance, fees, subjects, classes, sessions, and user roles.

The system is designed to provide a centralized platform for school administrators, teachers, staff, and students.

🚀 Project Overview

The School Management System helps schools manage their academic and administrative activities from a single application.

The system supports role-based access and provides separate functionality based on the user's permissions.

- Main Users
- Admin
- Teacher
- Staff
- Student
- Parent
